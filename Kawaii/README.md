### Installation
To use this theme, add `THEME_NAME` and `THEME_REPO` arguments to your `make` command:
```bash
$ make THEME_NAME=omega_kawaii THEME_REPO="https://github.com/Psi-Prod/Omega-Kawaii-Theme"
```
