### Installation
To use this theme, add `THEME_NAME` and `THEME_REPO` arguments to your `make` command:
```bash
$ make THEME_NAME=omega_freenumworks THEME_REPO="https://github.com/PierreDiab/Omega-FreeNumworks"
```
