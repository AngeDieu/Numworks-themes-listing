### Installation
To use this theme, add `THEME_NAME` and `THEME_REPO` arguments to your `make` command:
```bash
$ make THEME_NAME=solarized_dark THEME_REPO="https://github.com/orakless/Omega-Theme-Solarized-Dark"
```
