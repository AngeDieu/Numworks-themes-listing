### Installation
To use this theme, add `THEME_NAME` and `THEME_REPO` arguments to your `make` command:
```bash
$ make THEME_NAME={nsi_light/dark} THEME_REPO="https://github.com/Warzazat/Omega-Theme-Example"
```
