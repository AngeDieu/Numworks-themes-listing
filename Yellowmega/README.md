### Installation
To use this theme, add the `THEME_NAME` and `THEME_REPO` arguments to your `make` command:
```bash
$ make THEME_NAME={yellowmega_light/dark} THEME_REPO="https://github.com/Laporte12974/Yellowmega_Theme"
```
