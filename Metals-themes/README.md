### Installation
To use this theme, add `THEME_NAME` and `THEME_REPO` arguments to your `make` command:
```bash
$ make THEME_NAME={chrome_light/dark or gold or metal} THEME_REPO="https://github.com/Shadow15510/Omega_Themes"
```
